# E0da E0da

## How do I install these formulae?

`brew install e0da/e0da/<formula>`

Or `brew tap e0da/e0da` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "e0da/e0da"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
